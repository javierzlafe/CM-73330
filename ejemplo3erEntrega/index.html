Bienvenido a la Entrega 3

Este proyecto demuestra el uso combinado de HTML semántico, Bootstrap y SCSS.

Es completamente responsive y fue desarrollado aplicando buenas prácticas en la estructura del código y control de versiones con Git.

Estructura Semántica

Se utilizaron etiquetas HTML5 como <header>, <main>, <section> y <footer> para mejorar la accesibilidad y la organización del contenido.

SCSS con Variables

Se definieron variables de color, fuentes y espacios para facilitar el mantenimiento del estilo general.

Responsive con Media Queries

Se aplicaron media queries y unidades relativas para que el sitio se adapte correctamente a diferentes dispositivos (mobile, tablet y desktop).
